Documentation

Required files

Before you can use the script generator, you must have two files. A .srt file containing subtitles and a .txt file containing a movie script. The subtitles can be found at OpenSubtitles and the script at IMSDb or go to our search page.


The subtitles

You can search on our website for subtitles on on OpenSubtitles. When you have entered your search query, a list with movies will appear. Click on the movie you would like the subtitles of. Again a list will appear, but this time with the subtitles that are available for that movie. Click on the download button (the fifth column) of the subtitles you want. A zip file gets downloaded. Unzip the zip file on your computer and open the folder that will appear. The .srt file is the one that must be uploaded to the script generator.


The script

When getting your script from IMSDb you have to copy paste the text into a text file, because there is no download button. This text file should be uploaded to the script generator. You can also copy the URL of the script and paste that URL into the script generator after clicking “use URL” (you may have to press the button twice in order for it to show a text field to paste the URL in).


Uploading files

If you uploaded a URL for the script you only have to follow these steps for the subtitles.

When you have the files on your computer, you can upload them to the script generator. Upload the subtitles file in the first input field and the script file in the second. The files are uploaded correctly when the name of the file is shown beneath the button displaying ”upload your subtitles/script here”. Please make sure that the subtitles and the script are from the same movie, otherwise the program won’t work the way it should. After providing both the subtitles and script, click submit.


The results

Once the files are submitted and processed, you are taken to another page where the results are displayed. A JSON file is ready to be downloaded as well.
