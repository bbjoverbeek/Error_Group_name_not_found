Documentation

Required files

Before you can use the script generator, you must have downloaded two files. A .srt file containing subtitles and a .txt file containing a movie script. These can be found at OpenSubtitles for the subtitles and at IMSDb for the script files (other sides can be used as well, however, you do have to use the right type of files).

Uploading files

When you have downloaded the files onto your computer, you can upload them to the script generator. Upload the subtitles file in the first input field and the script file in the second. The files are correctly uploaded when the name of the file is shown beneath the button displaying "upload your subtitles/script here". Please make sure that the subtitles and the script are from the same movie, otherwise the program won't work the way it should. After uploading the files, submit the files.

The results

Once the files are submitted and processed, you are taken to another page where the results are displayed. A JSON file is ready to be downloaded as well.
