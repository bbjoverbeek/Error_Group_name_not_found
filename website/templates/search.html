Search Subtitles

After pressing enter, your search results are displayed on the website of opensubtitles. For more information how to get the subtitles, visit the documentation.

Search Script

When pressing enter on your keyboard your search query is sent to IMSDb. The movies scripts containing your search query are shown. For more information visit the documentation.

What to do after finding the movies

On OpenSubtitles you can download a .srt file. On IMSDb you cannot simply download a text file, unfortunately. You can open a webpage containing the script, but you cannot download it as a text file file. In order to get a text file you have to copy and paste the text. This file can be uploaded to our website. However, you can use the URL of the script as well. Copy that URL from IMSDb and go to the script generator. Clicking the "use URL" button will make a text field appear (sometimes you have to click twice though). Paste the URL in the text field.
